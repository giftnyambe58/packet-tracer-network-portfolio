**OBJECTIVES**
- putting vlans on separate access switches
- configuring trunk ports on access switches
- creating vlans on Multilayer switch
- configuring gateway for the vlans ( 10, 20, 30 and 40)
- DHCP configuration
- configuring routing protocol
- configuring static route
- turnining switch interface on MLS int a layer 3 interface(interface to the edge router)
- NAT for inside host (PAT)
- NAT for the server (Static)
  
**TECHNOLOGY USED**
- vlan
- ip routing
- DHCP
- NAT (PAT)
- NAT (Static)
- OSPF
- Static Route
